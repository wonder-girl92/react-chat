### Десктопный чат

### Запуск проекта

Для запуска проекта вам необходимо набрать команду в терминале:

` npm i `

После набрать команду:

`npm start`

### Основной функционал приложения:

- Отправка сообщений
- Удаление сообщений 
- Поиск сообщений
- Выбор чата
- Поиск контактов
- Профиль активного чата
